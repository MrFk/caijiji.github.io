如何快速找到可能拿下的目标

在做测试的时候 有时候面对一堆子域名会很头疼0.0 而且很多子域名都有诱惑性 比如*.jd.com *.taobao.com 很多子域名都是电商 这种子域名毫无利用价值0.0这个时候需要快速排查 分享一下自己的办法 大牛有新方法也可以留言一下>..<学习学习

用到的插件 https://github.com/h4cc/wkhtmltoimage-amd64

之后git clone一下 其实只需要bin目录下的wkhtmltoimage-amd64

Ubuntu

git clone https://github.com/h4cc/wkhtmltoimage-amd64
sudo apt-get install openssl build-essential xorg libssl-dev libxrender-dev  ttf-arphic-bsmi00lp
mv wkhtmltoimage-amd64 /usr/local/sbin/

Centos

yum install -y xorg-x11-fonts-75dpi
yum install -y xorg-x11-fonts-Type1
wget http://downloads.sourceforge.net/project/wkhtmltopdf/0.12.2.1/wkhtmltox-0.12.2.1_linux-centos7-amd64.rpm
rpm -Uvh wkhtmltox-0.12.2.1_linux-centos7-amd64.rpm

然后在终端输入 wkhtmltoimage-amd64看看成功运行不(Centos wkhtmltoimage)

提供一个check的Python


import os



def check(url):

    filename = url+'.png'

    #print filename

    cmd = 'wkhtmltoimage-amd64 -n %s %s' % (url, filename)

    #print cmd

    os.system(cmd)

if __name__  == '__main__':

    fo = open('jd.txt','r')

    # http://www.jd.com

    mubiao = fo.read().split('\n')

    for i in range(len(mubiao)):

        url = mubiao[i]

        check(url)
